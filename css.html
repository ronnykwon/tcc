Le CSS

A quoi ça sert?

Un aperçu rapide de la page de garde sans CSS

Le principe

         
            tag | class | selecteur {
               attribut: valeur;
            }

Le style est écrit dans un fichier portant l'extension .css

Exemple

         asset/css/styles.css
         
            body {
               background: green;
               color: white;
            }
         

         
            <!doctype html>
            <html>
               <head>
                  <meta charset="UTF8"/>
                  <link rel="stylesheet" type="text/css" href="asset/css/styles.css"/>
               </head>

               <body>
                  Ma première Page!
               </body>
            </html>
