Le Javascript

A quoi ça sert?

Le javascript est un language de programmation. Il permet d'écrire des scripts pour :

  • Manipuler le DOM
  • Dynamiser les pages

Le principe

         
            Il faut apprendre le langage Javascript! :)

Le script est écrit dans un fichier portant l'extension .js
Ou bien, le script peut être directement inclus dans le corps HTML.

Exemple

         
            <!doctype html>
            <html>
               <head>
                  <meta charset="UTF8"/>
                  <script src="asset/js/script.js"></script>
               </head>

               <body>
                  Ma première Page!
               
               <script>
               function somme() {   
                  var a = parseInt(document.getElementById('number-a').value);
                  var b = parseInt(document.getElementById('number-b').value);
                  if( isNaN(a) || isNaN(b) ) {
                     alert('valeurs incorrectes! Veuillez entrer des nombres!');
                     document.getElementById('number-a').value = 0;
                     document.getElementById('number-b').value = 0;   
                  }
                  document.getElementById('result').innerHTML = 'Résultat : ' + (a+b);
                  document.getElementById('result').classList.add('animate');
                  document.getElementById('result').classList.add('bounceIn');

               }
               </script>
               <input id="number-a"/>
               <input id="number-b"/>
               <button type="button" onclick="somme();">Calculer la somme !</button>
               <div id="result"></div>
               
            </body>
            </html>
